# 首页 Figma 设计稿 brief v1

> **状态**：交给设计师/AI 生成稿的"设计需求文档"
> **对应 implementation-plan-v1.md 第 12 章 Step 6 + 第 7.1 节首页模块**
> **目标尺寸**：桌面 1440px / 移动 375px 双稿

---

## 1. 设计目标

让访客在 **30 秒内** 完成 3 个判断：

1. **这是什么？** → 一个开箱即用、面向中国和外贸用户的内容管理系统
2. **能解决我什么？** → 不用懂代码也能搭官网/博客/产品站/外贸独立站
3. **要怎么开始？** → 看文档 / 立即试用，两个 CTA 一目了然

---

## 2. 整体调性

- **风格**：简洁、专业、克制（参考 Vercel / Resend / Linear）
- **不要**：花哨渐变、多余动画、塞满整屏的粒子效果
- **配色**（与本站后续 Tailwind 主题一致）：
  - 主色：Sky Blue `#0EA5E9`（与 BangNiCMS admin 主题一致）
  - 文字：`#0F172A`（slate-900）+ `#475569`（slate-600）
  - 背景：纯白 + `#F8FAFC`（slate-50）分区
  - 强调：`#F59E0B`（橙黄，仅用于 Final CTA 和价格强调）
- **字体**：
  - 中文：思源黑体 / PingFang SC / 系统默认
  - 英文：Inter
  - 代码：JetBrains Mono / Fira Code
- **圆角**：8px / 12px / 16px 三档
- **阴影**：仅卡片悬停时极轻 `shadow-sm`，不要厚重阴影

## 3. 首页区块（从上到下）

### 3.1 顶部导航栏 `Navbar` （高 64px，固定吸顶）

```
[Logo BangNiCMS] [产品][文档][扩展市场][博客]            [中/EN] [GitHub] [立即试用]
```

- 左：Logo + 文字
- 中：4 个一级导航
- 右：语言切换 + GitHub icon + 主 CTA 按钮（小尺寸版）
- 滚动 100px 后加底部 1px 边框分隔

### 3.2 Hero 区（高 ~600px）

**布局**：左 60% 文字 / 右 40% 产品截图（桌面）；移动堆叠

**文案草案**（中文版）：

```

  让中国人造出更好的网站

H2 副标题（20px 常规）

  10 分钟部署，0 代码运营，免费开源。

CTA 主按钮（实心 Sky Blue）
  立即免费试用 →

CTA 次按钮（描边）
  查看 5 分钟快速开始

底部 trust badges（弱化）
  ✓ 完全免费    ✓ 无需注册    ✓ 中文友好    ✓ 国内服务器友好
```

**右侧视觉**：
- 一张 admin 后台截图（带浏览器外框 mock）
- 或一段 30 秒产品演示 GIF（loop）

### 3.3 数据条 `Trust Strip`（高 80px，浅灰背景）

```
[ XX 个内容模型 ] [ XX 种语言 ] [ XX 个扩展 ] [ XX 用户部署 ]
```

数字用 Sky Blue 加粗，下方 12px 灰色 caption。上线初期可以低调（如 "4+/3+/12+/Beta 阶段"），不要造假数据。

### 3.4 核心特性 `Feature Grid`（3×2 或 4×2 卡片）

**8 个特性**（每个一张卡片：图标 + 标题 + 1 行描述）：

| 图标 | 标题 | 描述 |
|---|---|---|
| 🎨 | **内容模型可装可拆** | 产品/文章/下载 4 个内置模型 + 模型市场，按需扩展字段 |
| 🌐 | **多语言天生支持** | 多语言录入 + AI 翻译，外贸独立站直接可用 |
| 🧩 | **主题与插件机制** | 视觉换肤 + 功能扩展，无需改源码 |
| 🤖 | **AI 助手深度集成** | 翻译 / SEO 建议 / 内容改写，全部嵌在编辑流程里 |
| 🔍 | **SEO 全套就绪** | sitemap / 结构化数据 / meta 多语言，搜索引擎友好 |
| 🚀 | **一键部署** | Docker Compose 拉镜像即用，国内主流云全兼容 |
| 🛒 | **官方扩展市场** | 主题/插件/模型在线浏览安装，离线包也能装 |
| 💾 | **备份与升级有保障** | 自动备份 + 一键升级 + 一键回滚 |

卡片样式：白底 + 1px 边框 + 16px 内边距，hover 时边框变 Sky Blue。

### 3.7 快速开始 `Quick Start`

3 个 step，左侧编号 + 右侧代码块：

```
1. 拉镜像
   $ wget https://cms.bangni.com/docker-compose.yml
   $ docker compose pull

2. 启动
   $ docker compose up -d

3. 访问
   $ open https://你的域名.com/install
```

代码块带「一键复制」按钮（hover 显示）。
