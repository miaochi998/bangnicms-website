**BangNiCMS** 是一个**开源、自托管**的内容管理系统，专为**中文站长 / 外贸企业 / 内容创作者**设计。一句话概括：

> 用 10 分钟把一个**专业、多语言、可被搜索引擎收录**的官网或博客部署到自己的服务器上，从此不再依赖任何平台。

## 三个核心定位

## 谁适合用 BangNiCMS

- **外贸 / B2B / 工业品企业**：产品库 + 多语言询盘表单 + 谷歌 SEO 友好
- **跨境独立站起步阶段**：免月费、免抽成、数据完全自有
- **写作者 / 独立博客**：极简博客主题、文章工作流、RSS / 订阅
- **需要自定义内容结构的项目**：模型包扩展，无需改数据库 schema
- **追求性能与稳定**：服务端预渲染 + Caddy 自动 HTTPS + Lighthouse 90+

## 谁不适合用

- **完全不想接触服务器**：BangNiCMS 是自托管方案，你需要一台 VPS。
  如果完全不想运维，目前请用 Webflow / Wix / Shopify 等 SaaS。
- **重交易型电商**：本系统聚焦"官网 + 询盘 + 内容"，不内置购物车 / 在线支付 / 物流模块。
  如需完整电商能力，请用 Shopify / Magento / OpenCart。
- **需要 PHP 生态扩展**：本系统基于 Node.js / TypeScript，
  如果你有大量 WordPress 插件依赖，迁移成本较高。

```
[运营基础]
  工作台 · 站点设置 · 语言管理 · 用户管理

[内容创作]
  页面 · 产品 · 文章 · 下载 · 分类 · 菜单 · 媒体库

[扩展生态]
  扩展中心 · 页面组件 · 模型管理 · 主题配置（动态）

[转化与运营]
  询盘管理 · 数据统计 · AI 助手

[运维]
  系统升级 · 数据备份
```

这些模块的具体使用方式将在后续章节逐一介绍。
