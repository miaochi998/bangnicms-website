## 3. ICP 备案

### 3.1 备案前置条件

| 项 | 要求 |
|---|---|
| **服务器** | 国内云厂商（不能用海外服务器备案） |
| **域名** | 国内注册商持有 |
| **备案主体** | 个人 or 企业，需要法人/负责人身份证 |
| **联系电话** | 工信部短信核验，必须能正常接收 |
| **邮寄地址** | 部分省份要求邮寄幕布拍照（拍完寄回不丢就行） |

### 3.2 备案流程（以腾讯云为例）

1. 登录腾讯云 → 备案管理 → 新增备案
2. 填写主体信息（个人则填法人身份证 / 企业填营业执照）
3. 填写网站信息：
   - 网站名称：「BangNiCMS 官方文档」（不能含"官方""中国""政府"等敏感词）
   - 网站负责人：（同主体）
   - 网站内容：「内容管理系统介绍与文档」
4. 上传材料：
   - 身份证正反面 / 营业执照
   - 法人手持身份证人脸识别（小程序拍照）
   - 网站负责人核验单（PDF 在线签）
5. 等待初审（1-3 工作日）
6. 工信部审核（5-15 工作日）
7. 通过后会下发 ICP 备案号（如 `京ICP备2026XXXXXX号-1`），需在网站页脚展示

### 3.3 备案期间过渡方案

备案需要 1-2 周，期间国内服务器**不能开 80/443**对外服务。两个选择：

- **A**：海外 staging 服务器先跑（如 Vultr 东京 / DigitalOcean 新加坡，¥40-60/月）
  - `staging.cms.bangni.com` 解析到海外 IP
  - 备案期间团队内部测试
- **B**：在本地 Docker 跑，仅本机访问
  - 不烧服务器钱
  - 但无法对外分享演示

> **建议**：A，¥50/月烧 1 个月即可。海外 staging 上线后顺便做 Week 1 的"拉镜像跑通"验证。
