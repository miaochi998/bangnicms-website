走完安装向导后，你已经有一个**可登录的 BangNiCMS 实例**，但还需要完成 10 件事才能正式投入运营。本文按优先级逐一带你做完，**总耗时约 30-60 分钟**。

<Aside type="tip">
本文每一步都对应后台菜单中的某一项。如果你想跳过某些步骤后续再做，**优先级 P1** 的请务必完成，**P2/P3** 可后续补。
</Aside>

## P1：必做的 5 件事

### 1. 品牌信息（LOGO / 主色 / 名称）

**入口**：「**站点设置**」→ 品牌（顶部第一个 Tab）

| 字段 | 说明 | 建议 |
|---|---|---|
| `siteName` | 站点名称 | 如"邦尼网络"（已在向导第 4 步填过，可改） |
| `siteSubtitle` | 站点副标题 | 如"专注外贸独立站建设"（可选） |
| `siteDescription` | 站点描述 | 用于 SEO meta description（150 字内） |
| `siteLogo` | LOGO 图片 | 推荐 SVG 或 200×60 PNG，透明底 |
| `siteFavicon` | 浏览器图标 | 32×32 PNG/ICO |

<Aside type="note" title="主品牌色">
品牌主色由**当前激活主题**的 `config.primaryColor` 控制，请到「**主题配置**」（Sidebar 折叠组）调整，而非「站点设置」。
</Aside>

### 2. SEO 默认（标题模板 / 描述 / 关键词）

**入口**：「**站点设置**」→ SEO 默认

| 字段 | 说明 | 示例 |
|---|---|---|
| `siteDescription` | 默认 meta description | "邦尼网络 - 跨境独立站一站式建站方案" |
| `siteKeywords` | 默认 meta keywords | "外贸建站, 独立站, 多语言 CMS"（逗号分隔） |

每条产品 / 文章可在编辑页**单独覆盖** SEO 字段；未填则继承这里的默认值。

<Aside type="tip">
如安装了 SEO 工具集插件，会额外提供「**实时 SEO 评分**」与「**结构化数据自动注入**」能力。详见 [SEO 工具集](/extensions/plugin-seo-toolkit)。
</Aside>

### 3. 启用所需语言

**入口**：「**语言管理**」

安装向导默认只启用了一个**默认语言**。如需多语言：

<Steps>

1. 「语言管理」→ **新增语言**
2. 选择语言代码（如 `en-US` / `es-ES`）
3. 填语言显示名（如 "English" / "Español"）
4. 选择是否启用
5. 保存

</Steps>

新增后，所有 4 类内容（产品/文章/下载/页面）的编辑页会自动出现该语言的 Tab。

<Aside type="note">
**默认语言**只能有一个，可在「语言管理」点击其它语言旁的"设为默认"切换。**注意**：切换默认语言会改变 URL 结构（取决于 `urlStrategy`），请在内容尚少时操作。
</Aside>

### 4. 上传与媒体策略

**入口**：「**站点设置**」→ 系统 → 上传与媒体设置

| 字段 | 默认值 | 建议 |
|---|---|---|
| `maxUploadSize` | 10 MB | 视频较多时调到 50-100 MB |
| `maxVideoSize` | 50 MB | 高清视频站点可调 200 MB |
| `allowedUploadTypes` | jpg/png/webp/pdf/zip 等 | 按需新增（如 `mp4` `mov` 视频扩展） |
| `imageCompression` | 启用 | 推荐保持 |
| `fileNamingRule` | 默认（hash） | hash 命名避免冲突 |
| `videoThumbnailEnabled` | 启用 | 视频自动取第一帧做封面 |

<Aside type="caution">
**调整后立即对全站生效**——所有上传组件（媒体库、富文本插图、产品图集等）都会读取这里的策略。
</Aside>

### 5. 配置 header / footer 菜单

**入口**：「**菜单管理**」

如果你在向导第 5 步选了**导入示例数据**，菜单已有预设；否则需要手动创建：

<Steps>

1. 「菜单管理」→ **新增菜单**
2. 填 `key`：`header`（主导航）
3. 添加菜单项：
   - 类型：自定义 URL / 内容（产品/文章/...）/ 分类
   - 多语言标签
   - 排序
4. 保存
5. 重复创建 `key=footer`（页脚菜单）

</Steps>

### 在主题里指定菜单位置

主题（如 official-site）会在模板里通过 `key` 调用菜单。例如：

- 主题 header 调用 `key=header` 的菜单
- 主题 footer 调用 `key=footer` 的菜单

详见 [主题（即将上线）](#)。

## P2：建议尽快做的 3 件事

### 6. 修改超管账号信息

**入口**：「**用户管理**」→ 自己的账号

向导创建的超管可补充 / 修改：

- 邮箱（用于密码找回）
- 显示名
- 头像

如忘记密码：在 SSH 服务器上执行

```bash
docker exec -it bangnicms-server pnpm reset-password admin
```

### 7. 接入 AI 助手

**入口**：「**AI 助手**」→ 配置

支持接入 OpenAI / Claude / Doubao 等 API：

| 字段 | 说明 |
|---|---|
| Provider | 选 OpenAI / Claude / Doubao |
| API Key | 你的 provider API key |
| Base URL | 自部署或代理时填，否则空 |
| Model | 如 `gpt-4o-mini` / `claude-3-5-sonnet-20241022` |

**测试连接** → 成功后启用。

启用后**自动获得**：

- 媒体上传时**自动生成 alt 描述**（图片）
- 多语言批量翻译（语言管理 → 待翻译队列）
- 内容辅助创作

详见 [AI 助手（即将上线）](#)。

### 8. 录入第一条业务内容

**入口**：「**产品管理**」 / 「**文章管理**」

至少建议录入：

- **3 件产品**（如做外贸 B2B）
- **2 篇公司新闻 / 行业洞察**（建立专业感）

每条内容都填写：

- 默认语言版本（必填）
- 至少一个非默认语言版本（如有多语言）
- SEO 字段（标题 / 描述）
- 封面图、详情正文
- 状态切到 **已发布**

## P3：可后续补的 2 件事

### 9. 配置自动备份

**入口**：「**数据备份**」→ 备份计划

| 字段 | 推荐 |
|---|---|
| 启用 | ✓ |
| Cron 表达式 | `0 3 * * *`（每天凌晨 3 点） |
| 保留份数 | 7 |
| 备份目标 | 服务器本地 / S3 / OSS |

详见 [数据备份（即将上线）](#)。

### 10. 验证前台与 SEO

最后一步：以**访客视角**走一遍前台。

- 访问 `https://你的域名`
- 检查首页布局、菜单、Hero、CTA 是否正确
- 点几条产品 / 文章 → 检查详情页布局
- 提交一条**测试询盘** → 后台「询盘管理」应能看到
- 浏览器 → 查看页面源代码 → 检查：
  - `<title>` 是否包含站点名 + 内容标题
  - `<meta name="description">` 不为空
  - `<link rel="alternate" hreflang="...">` 多语言时存在
- 用 [PageSpeed Insights](https://pagespeed.web.dev/) 跑一遍 → 期望 Lighthouse 桌面端 90+

## 完成 🎉

如果以上 10 件事全部完成，你的 BangNiCMS 站点已经**正式上线**，可以投入运营。
